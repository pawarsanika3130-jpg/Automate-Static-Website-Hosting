# Automate Static Website Hosting using AWS S3 and Python boto3

## Project Overview

This project automates the deployment of a static website to Amazon S3 using Python and the boto3 SDK.

Instead of manually creating buckets and uploading files through the AWS Console, the entire process is automated using code.

## AWS Services Used

* Amazon S3
* AWS IAM
* AWS CLI

## Technologies

* Python
* boto3
* HTML/CSS

## Features

* Create and manage S3 buckets
* Upload website files automatically
* Enable static website hosting
* Host a public website on AWS
* Infrastructure automation using Python

## Architecture

Python Script → boto3 → Amazon S3 → Static Website Hosting → Public Website URL
